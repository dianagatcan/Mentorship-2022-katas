### 10 Is it a triangle?

Implement a method that accepts 3 integer values a, b, c. The method should return true if a triangle can be built with the sides of given length and false in any other case.

#### Examples

```
isItATriangle(3, 3, 3)
  => true
```

```
isItATriangle(4, 3, 10)
  => false
```
